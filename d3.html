核心概念 state vuex 使用的是单一状态书 单一状态树的意思是用一个对象就包含了全部应用层级状态，这样的话可以看出，它可以作为一个唯一数据源来存在 每个项目都仅仅包含一个store实例，单一状态树可以直接定位任一特定的状态片段，在调试的过程中也能轻易取出当前应用的状态 Vue从vuex中读取状态值的方法 1、可以在计算属性中直接从store中读取 eg: computed: { coun() { return store.state.count } } 这种方式的缺点就是，每个需要使用全局store的组件，都得需要频繁引入，这样比较麻烦 2、vuex 提供了store选项，可以将store实例注入到vue根组件上（如果按照之前单独使用的话都得引入Vue.use(vuex)），这样所有的组件都不需要再次引入，直接使用就行
